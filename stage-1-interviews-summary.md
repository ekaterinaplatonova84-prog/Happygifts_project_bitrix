# Этап 1: Интервью с ключевыми сотрудниками — итог диалога

**Дата:** 28 июня 2026
**Проект:** Наведение порядка в Битрикс24 и прозрачная аналитика для Happy Gifts Group

## Контекст задачи

- Битрикс24 внедрён, но почти не используется; процессы хаотичные.
- Менеджеры по продажам ведут сделки в Excel.
- Проект рассчитан примерно на год.
- Заказчик — новый директор по маркетингу.
- Цель: навести порядок в бизнес-процессах для роста производительности и прозрачной аналитики.

## Первый этап

Проведение интервью с ключевыми сотрудниками: директор по маркетингу, РОПы, менеджеры по продажам и сопровождению. Структура компании пока известна не полностью.

## Что изучено о компании

- Рекламно-сувенирная продукция, корпоративные подарки (B2B), 28+ лет на рынке.
- Головной офис в Москве + 19 офисов продаж + партнёрская сеть (РФ и ближнее зарубежье).
- Ассортимент 20000+ позиций.
- Есть собственное производство (нанесение логотипов): сделка = товар + макет + нанесение + резерв + доставка + договор.
- Каналы: онлайн-каталог с ЛК, email-маркетинг (UniSender, SendPulse, Mailchimp), Telegram/VK.

**Вывод:** не классическая воронка «лид → продажа», а цепочка **Продажи → Производство → Логистика/Сопровождение** с филиальной структурой. Интервью должно вскрыть стыки между звеньями.

## Формат сбора информации

Согласованы два формата:
- **Видеосозвоны 1-на-1** — для тех, кто может выделить время.
- **Асинхронные опросники** — для занятых сотрудников.

## Следующие шаги

1. Провести интервью / собрать заполненные опросники.
2. Свести ответы в карту процессов «как есть» (As-Is).
3. Выявить узкие места и причины неиспользования Битрикс24.
4. Сформировать карту «как должно быть» (To-Be) и согласовать с заказчиком.
5. Перейти к настройке воронок, полей, бизнес-процессов и аналитики.
