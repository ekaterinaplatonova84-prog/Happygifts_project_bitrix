# BI Дашборд для Битрикс24 — обсуждение

**Дата:** 28 июня 2026

## Вопрос: Возможность и способ деплоя

**User:** возможно ли написать BI дашборд для Битрикс24? Как его задеплоить в сам Битрикс? Нужен ключ?

**Ответ:** Да, задача реализуема. Есть 3 варианта интеграции:

### Вариант 1: Внешний дашборд (рекомендуется для старта)

- Размещается на своём сервере / Vercel / Netlify как обычное веб-приложение.
- Bitrix24 отдаёт данные через REST API.
- Достаточно **вебхука** (создаётся в настройках Bitrix24: *Приложения → Вебхуки → Исходящий вебхук/REST API*).
- **Ключ приложения не требуется.**
- **Плюсы:** минимальный порог входа, простота деплоя, не зависит от маркетплейса.

### Вариант 2: Встроенный iframe-виджет

- Регистрируется как локальное приложение в Bitrix24 Marketplace.
- Открывается прямо внутри интерфейса Битрикс24.
- Нужен **ключ приложения** (client_id / client_secret), который генерируется при регистрации.
- **Плюсы:** глубоко интегрирован в интерфейс.

### Вариант 3: Самописный отчёт в разделе «Аналитика»

- Использует встроенную BI-систему Битрикс24.
- **Минусы:** ограниченный функционал, полноценный кастомный дашборд так не сделать.

### Решение

Начинаем с **Варианта 1** (внешний дашборд + REST API вебхук) — минимум порога входа.
